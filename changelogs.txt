<h3>v1.4.0 (Latest)</h3>
                            <ul>
                                <li><strong>remove.bg API:</strong> Added support for remove.bg Cloud API as an optional backend. When an API key is configured, the server uses remove.bg's gold-standard AI instead of local models — resulting in near-zero RAM usage and best-in-class quality.</li>
                                <li><strong>Session Recovery:</strong> Website now automatically restores your progress after your phone sleeps or you switch tabs. Remove BG results and active downloads are saved and resumed seamlessly when you return.</li>
                                <li><strong>Memory Optimization:</strong> Implemented OS-level memory trimming (malloc_trim) after each Remove BG request, ensuring freed memory is actually returned to the system and not held in Python's internal buffer pool.</li>
                                <li><strong>Server Stability:</strong> Added startup cleanup on server boot — any leftover AI-processed images from a previous session or crash are automatically detected and cleaned up, preventing data from being retained indefinitely.</li>
                            </ul>
